# 多表查询接口文档

## 1. 根据业主ID查询仪表信息和仪表配置信息

- **接口地址**: `/api/complex/owner/{ownerId}/meters`
- **请求方法**: `GET`
- **接口描述**: 根据业主ID查询该业主的仪表信息和仪表配置信息
- **请求参数**:
  - `ownerId`: 业主ID（必填）
- **返回结果**: 业主的仪表信息和仪表配置信息

## 2. 根据抄表ID查询抄表员信息和业主信息

- **接口地址**: `/api/complex/meterReading/{readingId}/details`
- **请求方法**: `GET`
- **接口描述**: 根据抄表ID查询抄表员信息和业主信息
- **请求参数**:
  - `readingId`: 抄表记录ID（必填）
- **返回结果**: 抄表员信息和业主信息

## 3. 根据抄表记录中的仪表ID查询仪表信息和仪表配置信息

- **接口地址**: `/api/complex/meterReading/meter/{meterId}/details`

- **接口描述**: 根据抄表记录中的仪表ID查询仪表信息和仪表配置信息
- **请求参数**:
  - `meterId`: 仪表ID（必填）
- **返回结果**: 仪表信息和仪表配置信息

## 6. 新增房屋业主关联

- **接口地址**: `/api/complex/houseOwner`
- **请求方法**: `POST`
- **接口描述**: 新增房屋业主关联，需要输入业主ID和房屋ID
- **请求参数**:
  - `ownerId`: 业主ID（必填）
  - `houseId`: 房屋ID（必填）
- **返回结果**: 操作结果
